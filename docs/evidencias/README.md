# Evidências da Implementação

Salve nesta pasta os prints e logs usados no PDF técnico.

## Evidências recomendadas

- `pipeline-sucesso.png`
- `pipeline-falha.png`
- `trivy-log.txt`
- `artefato-trivy.png`
- `pull-request-aprovado.png`

## O que cada evidência deve mostrar

### pipeline-sucesso.png

Print do GitHub Actions finalizado com sucesso.

### pipeline-falha.png

Print da simulação com falha causada pela presença de um arquivo `.env`.

### trivy-log.txt

Trecho do relatório do Trivy ou arquivo baixado como artefato no GitHub Actions.

### artefato-trivy.png

Print mostrando o artefato `trivy-security-report`.

### pull-request-aprovado.png

Print do Pull Request após o pipeline passar.
